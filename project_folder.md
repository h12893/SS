# population_job

人口の多寡や増減が、IT関係の業務数にどのように影響しているかの調査。

前提として
- 人口が多い都市部の方が、IT関係の企業や案件が多い
- 人口が少ない地方の方が、IT技術の導入による効率化の恩恵をより受けられる
という二つの考察があり、それらの検証も兼ねた調査とする。
また、[島根県の離島でのデータ活用ガイドツアー](https://rejoui.co.jp/report/5053/)や[茨城県立竜ケ崎第一高等学校での"デジ探"活動](https://www.ryugasaki1-h.ibk.ed.jp/learning/%E3%83%87%E3%83%BC%E3%82%BF%E3%83%BB%E3%83%87%E3%82%B8%E3%82%BF%E3%83%AB)などが[データサイエンティスト協会のシンポジウム](https://www.datascientist.or.jp/symp/2025/)で紹介されていたことも踏まえ、地方でのデータ活用が広がっているのかという観点も併せて調査を行う。

集計は年度単位で行い、業務数は有効求人数や就職件数を主に用いる。また、平均求人賃金などを追加情報として扱うことも想定される。

人口と業務はともに都道府県単位で集計する。
